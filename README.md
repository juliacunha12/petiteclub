# 🎀 petiteclub — Curated Petite Fashion

O **petiteclub** é uma plataforma de e-commerce de luxo minimalista, focada na curadoria de moda para quem busca o caimento perfeito (PP e P). Este projeto demonstra a união entre um design sofisticado e uma arquitetura escalável, utilizando as tecnologias mais modernas do ecossistema Fullstack.

## 🌟 O Diferencial

Ao contrário de e-commerces genéricos, o **petiteclub** foi construído sob o conceito de **"Experience First"**. Cada detalhe — da tipografia ao tempo de carregamento — foi planejado para transmitir exclusividade e elegância.

- **Interface Premium:** Design minimalista inspirado em boutiques de alta costura.
- **Arquitetura Baseada em Dados:** Gestão de inventário e categorias via Supabase com Row Level Security.
- **UX Fluida:** Navegação intuitiva com filtros dinâmicos e sacola de compras responsiva.

## 🚀 Stack Tecnológica

- **Frontend:** React + Vite (Custom CSS Architecture)
- **Backend:** Python + FastAPI (Serverless Ready)
- **Infrastructure:** Supabase (PostgreSQL & Auth)
- **Deployment:** Vercel

## 📐 Estrutura do Projeto

O projeto segue o padrão de monorepo moderno:
- `/frontend`: Aplicação Client-side com foco em performance.
- `/backend`: API robusta para orquestração de serviços.
- `/sql`: Camada de persistência e segurança de dados.
